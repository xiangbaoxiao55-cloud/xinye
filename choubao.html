🔮 Embedding API（记忆向量检索）
用于把记忆转成向量做语义检索，支持 OpenAI text-embedding-3-small。留空则自动用主API试，不支持时降级为关键词匹配。
GLM 用 embedding-3（URL填 https://open.bigmodel.cn/api/paas/v4/）；OpenAI 用 text-embedding-3-small；留空默认 text-embedding-3-small
👁️ 识图 API（图片先转文字再发主模型）
激活后填入下方字段，点「保存设置」生效
发图时先用便宜视觉模型描述成文字，主模型只收文字。描述会保留在历史记录里，之后几轮也能引用。
🎨 画图 API
激活后填入下方字段，点「保存设置」生效

关闭时等待完整回复（兼容性更好）；开启时逐字输出（更有沉浸感）
