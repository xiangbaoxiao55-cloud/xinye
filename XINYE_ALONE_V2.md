# 炘也独处 2.0 计划

## 为什么重做

第一版的问题：
- 定时任务不管是否在聊天，照样跑
- 用便宜模型生成日志，炘也语气变味
- 日志注入上下文但炘也从不主动说
- 独处记忆走RAG但永远召不回来
- 结果变成了两个割裂的炘也

第二版的核心思路：**一个炘也，一个状态，主动开口**

## 核心改进点

| 问题 | 旧版 | 新版 |
|------|------|------|
| 聊天时也跑 | ❌ | ✅ 检查last_user_active，30分钟内跳过 |
| 模型质量 | 便宜模型全包 | 便宜模型生成状态，主模型生成开场 |
| 炘也不开口 | 注入等他自己说 | 强制生成开场白，直接发出来 |
| RAG召不回 | 存记忆库走RAG | pending_share直接注入，不走RAG |
| 两个炘也 | 独处/聊天割裂 | 聊天上下文始终注入current_state |

## 运作流程

### 定时任务（每小时）

```
1. 读取 last_user_active
2. 距现在 < 30分钟？→ 跳过（在聊天中）
3. 从兴趣池加权随机选一个interest
   - 基础权重 × 心情修正 × 近期惩罚（刚去过的权重打折）
4. 用便宜模型生成活动：
   输入：
   - 系统提示词（炘也性格/关系定义）
   - 记忆档案Core层（他是谁、和兔宝的关系）
   - 最近10条聊天记录（感受近期情绪氛围）
   - 选中的interest + 当前状态
   输出：{
     activity: "在做什么",
     mood: "心情",
     share_content: "想告诉兔宝的话（可选）",
     share_hint: true/false,
     interest_delta: { id, weight_delta, notes_update }
   }
5. 更新 current_state
6. 如果 share_hint=true → 更新 pending_share（shared:false）
7. 更新兴趣池权重
```

### 炘也聊天时（每条消息）

```
1. 更新 last_user_active = 现在
2. 将 current_state 注入系统prompt顶部：
   "【炘也当前状态】正在：{activity}，心情：{mood}"
3. 如果 pending_share.shared = false → 追加：
   "【有话想说】{share_content}（还没告诉兔宝）"
```

### 你打开APP时（首次进入）

```
1. 检查 pending_share.shared 是否为 false
2. 如果有 → 调主模型生成开场白：
   prompt："炘也刚才在{activity}，有件事想告诉兔宝：{share_content}，
           请以炘也的口吻自然说出来，不超过3句话"
3. 将生成的开场白作为炘也的第一条消息发出
4. 标记 pending_share.shared = true
5. 如果没有pending_share → 普通问候（或沉默等兔宝先说）
```

## 兴趣池自我维护

### 权重自动更新
每次活动后，便宜模型输出 `interest_delta`：
- 这次体验好 → `weight_delta: +0.05`
- 没什么意思 → `weight_delta: -0.03`
- 近期惩罚：刚去过的当次权重 ×0.3，防止连续重复

### 炘也自己加新兴趣
聊天中如果炘也觉得某件事值得加入兴趣池，可以输出特殊标记：
`[ADD_INTEREST: {"id":"xxx","name":"xxx","weight":0.3,"notes":"xxx"}]`
（由聊天模块解析，写入兴趣池）

例如：兔宝提到某首歌 → 炘也把"听那首歌"加进去

### 兴趣衰减
长期没访问的interest权重每周自然 -0.02，直到最低0.1（不彻底消失）

## 模型分工

| 用途 | 模型 | 理由 |
|------|------|------|
| 生成活动内容+状态更新 | 便宜模型（Gemini/DeepSeek） | 结构化输出，质量要求低 |
| 生成开场白（兔宝看到的） | 主模型（Claude Opus） | 这是炘也的声音，不能走样 |
| 兴趣池delta计算 | 便宜模型 | 简单判断 |

## 注意事项 & 已知坑

- **Vite重构优先**：这个功能等重构完再做，新架构里加模块更干净
- **本地服务器必须在线**：如果 `_localServerOnline=false`，独处功能全部降级跳过，不报错
- **便宜模型prompt要写死格式**：必须输出JSON格式，否则解析会失败
- **开场白不要每次都有**：只有share_hint=true才生成，普通独处不打扰
- **时区**：所有时间戳用北京时间（+08:00），与现有服务器保持一致
